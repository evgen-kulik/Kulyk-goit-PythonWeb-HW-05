Пояснення до домашнього завдання з модуля №5.
1. Обов'язкова частина
Реалізована в main.py.

Приклад введення команди в термінал: python main.py 2

Приклад виводу в терміналі:
[{'08.07.2023': {'EUR': {'sale': 40.85, 'purchase': 39.85}, 'USD': {'sale': 37.2, 'purchase': 36.6}}},
{'09.07.2023': {'EUR': {'sale': 40.85, 'purchase': 39.85}, 'USD': {'sale': 37.2, 'purchase': 36.6}}}]

2. Додаткова частина
2.1. Додано можливість виведення курсу додаткових валют ("PLZ", "GBP", "CHF").
Реалізована в python main_different_currencies.py.

Приклад введення команди в термінал: python main_different_currencies.py -am 2 -cur PLZ

Приклад виводу в терміналі:
[{'09.07.2023': {'EUR': {'sale': 40.85, 'purchase': 39.85}, 'USD': {'sale': 37.2, 'purchase': 36.6},
'PLZ': {'sale': 9.006, 'purchase': 8.8884}}}, {'10.07.2023': {'EUR': {'sale': 40.85, 'purchase': 39.85},
'USD': {'sale': 37.2, 'purchase': 36.6}, 'PLZ': {'sale': 9.006, 'purchase': 8.8884}}}]

2.2. Зроблено чат (набір файлів в теці "chat_hw").
При введенні в поле "message" команди "exchange", виводиться курс валют станом на сьогодні для EUR та USD.
При введенні в поле "message" команди "exchange 2" (чи іншої кількості днів - не більше 10),
виводиться курс валют для вказаної кількості останніх днів для EUR та USD.
Кожен запит курсу валют в чаті логується в файл exchange.log.
